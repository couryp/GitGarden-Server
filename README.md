# Git Galaxy

Git Galaxy is a React browser application that visually represents a user's recent code on Github. After logging in with OAuth, Git Galaxy will retrieve commits from your most recently updated repositories. Then, these commits are piped through IBM's Watson Tone Analyzer and displayed on a three.js canvas. The purpose of Git Galaxy is twofold: first, to encourage better Git commit behavior; second, to display information in an exciting fashion.
